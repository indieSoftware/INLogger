LogFileType

public enum LogFileType

Points to a specific log file type and defines what to return when requesting for the log file’s content.

  • logFile

    Points to the current log file, therefore, only the log file’s content is returned. The backup file is ignored.

    Declaration

    Swift

    case logFile
  • backupFile

    Points to the backup file, therefore, only the backup file’s content is returned. The current log file itself is ignored.
