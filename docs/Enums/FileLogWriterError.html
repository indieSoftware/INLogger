FileLogWriterError

public enum FileLogWriterError : Error, CustomStringConvertible, Sendable

Errors which can arise in the FileLogWriter, but can’t be logged nor should they be thrown. To get informed of them assign a custom closure to the writer’s static variable logInternalError.
