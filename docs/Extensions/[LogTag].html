[LogTag]

  • highestPriorityLogTagState

    Returns the LogTagState with the highest priority of all LogTag states in this list.

    The highest priority is determined by comparing the states of all tags and returning the one with the highest priority. The order is provided by the LogTagState‘s Comparable order, which is as follows (highest first): forceDisabled > enabled > disabled

    However, when the array is empty then enabled will be returned.

    Declaration

    Swift

    var highestPriorityLogTagState: LogTagState { get }
